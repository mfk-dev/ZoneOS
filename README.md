# This Is ZoneOS

## What is ZoneOS?
- ZoneOS is an very simple, not finished yet All-In-One tool.
- The name ZoneOS is just for fun, it isnt an OS obviously.

## What does it have?
** For Now **
- 1. DNS Lookup            ( ** Available ** )                                                                                                                                                                                 
- 2. Port Scanner          ( ** Not Available For Now ** )                                                                                                                                                                     
- 3. Subdomain Scanner     ( ** Not Available For Now ** )                                                                                                                                                                     
- 4. Ping                  ( ** Not Available For Now ** )                                                                                                                                                                    
- 5. Website Information   ( ** Not Available For Now ** )                                                                                                                                                                     
- 6. Hash Generator       ( ** Available ** )                                                                                                                                                                                 
- 7. Password Generator    ( ** Available ** )

## Requirements
- In order to run the zoneos.py, you should download these libraries:
- rich
- dnspython
- To download them, enter these commands on a terminal:
- pip install rich
- pip install dnspython
